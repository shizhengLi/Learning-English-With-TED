# 09 防守技术：从盯人到协防

## Opening Scene

> "He's an **elite on-ball defender** with **active hands**, **great lateral quickness**, and the **basketball IQ** to read the offense. He can **switch** onto anyone, **fight over screens**, and provide **help-side defense** without **losing his man**."

防守是 NBA 最被低估的技术，也是英文篮球术语最密集的领域。这一篇带你搞懂从单防到团队防守的全套词汇。

## 防守体系 Defensive System

```
单兵防守 (Individual Defense)
    ↓
团队防守 (Team Defense)
    ↓
防守策略 (Defensive Schemes)
```

## 一、单兵防守 Individual Defense

### 基础防守动作

| 英文             | 音标               | 中译     | 说明         |
| ---------------- | ------------------ | -------- | ------------ |
| on-ball defense  | /ɑːn bɔːl dɪˈfens/ | 持球防守 | 防守持球人   |
| off-ball defense | /ɔːf bɔːl dɪˈfens/ | 无球防守 | 防守无球队员 |
| man-to-man       | /mæn tuː mæn/      | 人盯人   | 一对一防守   |
| defensive stance | /dɪˈfensɪv stæns/  | 防守姿势 | 防守基本站姿 |
| stay in front    | /steɪ ɪn frʌnt/    | 保持在前 | 不被过掉     |
| contain          | /kənˈteɪn/         | 限制     | 控制进攻球员 |
| slide            | /slaɪd/            | 横移     | 防守脚步移动 |
| shuffle          | /ˈʃʌfl/            | 碎步移动 | 防守碎步     |

**基本要求**：
> "Stay **low**, **hands up**, **feet moving**."
> 重心低、手举起、脚步动。

这是防守的三大基本原则。

**固定表达**：
- **get into him** = 贴身防守
- **stay attached** = 紧贴不放
- **lock down** = 完全锁死
- **clamp down** = 严防死守

> "He can **lock down** the opposing team's best player."
> 他能锁死对方的最佳球员。

### 防守手部技术

| 英文          | 音标           | 中译     | 说明               |
| ------------- | -------------- | -------- | ------------------ |
| active hands  | /ˈæktɪv hændz/ | 活跃的手 | 不断干扰的手部动作 |
| deflection    | /dɪˈflekʃn/    | 碰球     | 触碰到球改变路线   |
| strip         | /strɪp/        | 掏球     | 把球从对手手中打掉 |
| poke away     | /poʊk əˈweɪ/   | 捅掉     | 把球捅走           |
| reach-in      | /riːtʃ ɪn/     | 伸手掏球 | （容易犯规）       |
| hand-checking | /hænd ˈtʃekɪŋ/ | 手部接触 | 用手推对手（犯规） |
| contest       | /kənˈtest/     | 干扰     | 干扰投篮           |

> "He's got **active hands** and leads the league in **deflections**."
> 他手部很活跃，碰球数联盟第一。

**Contest 的重要性**：
> "You've got to **contest every shot**."
> 你必须干扰每一次投篮。

`contest` 作动词时特指**防守时举手干扰投篮**，这是防守的基本原则。

### 防守强度描述

| 表达                    | 含义               |
| ----------------------- | ------------------ |
| **lockdown defender**   | 锁死型防守者       |
| **pesky defender**      | 烦人的防守者       |
| **physical defender**   | 身体对抗强的防守者 |
| **disruptive defender** | 破坏性防守者       |
| **versatile defender**  | 多面手防守者       |
| **elite defender**      | 顶级防守者         |

> "Kawhi is a **two-way player** — elite on both ends."
> 卡哇伊是双向球员——攻防两端都是顶级。

`two-way player` 这里不是"双向合同"，而是指**攻防两端都强**的球员。

| ---------- | ------------- | ------ | ---------------- |

| hedge      | /hedʒ/        | 延阻   | 暂时夹击后回位   |

**Close out 的技术要点**：
> "**Close out with high hands** and **short steps**."
> 扑防时手举高，步伐要小。

这是防止投手轻松出手和突破的技术要领。

**Switch 的应用**：
> "They **switch everything** on defense."
> 他们防守时所有掩护都换防。

`switch everything` = 无限换防（现代篮球流行策略）

## Listen & Learn

一段防守解说：

> "Beautiful **help-side rotation** here. The **on-ball defender** **fights over the screen**, the big man **hedges hard**, and when the ball handler **attacks downhill**, the **weak-side defender** **rotates over** for the **block**. Then they **recover** perfectly. That's **championship-level defense**."

拆解：
- **fights over the screen** = 绕过掩护
- **hedges hard** = 用力延阻
- **attacks downhill** = 下坡式进攻
- **rotates over** = 轮转过来
- **championship-level** = 冠军级别

翻译：这次协防侧轮转非常漂亮。持球防守人绕过掩护，大个用力延阻，当持球人发起下坡式进攻时，弱侧防守人轮转过来完成封盖。然后他们完美回防。这就是冠军级别的防守。

## Practice 小练习

**Ⅰ. 场景匹配**

匹配防守术语与场景：
1. 投手准备出手，你快速扑上去 → ____
2. 队友被过，你从侧面补防 → ____
3. 掩护来了，你从上方挤过去 → ____
4. 你防的人传球了，你回到自己的人身边 → ____

选项：recover / close out / help defense / fight over

**Ⅱ. 翻译**

1. He's a lockdown defender with active hands and great lateral quickness.
2. They switch everything to avoid getting caught on screens.
3. He provides elite rim protection as the last line of defense.

**Ⅲ. 填空**

1. You've got to ____ every shot, even if you can't block it.
2. Stay ____, hands up, feet moving.
3. The weak-side defender needs to ____ over when the ball penetrates.

---

**参考答案**

- Ⅰ. close out / help defense / fight over / recover

- Ⅱ.
  1. 他是锁死型防守者，手部活跃，横向移动速度极佳。
  2. 他们所有掩护都换防，避免被困住。
  3. 他作为最后防线提供顶级护筐。

- Ⅲ. contest / low / rotate
