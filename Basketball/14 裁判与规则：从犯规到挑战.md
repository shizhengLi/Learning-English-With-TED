# 14 裁判与规则：从犯规到挑战

## Opening Scene

> "Whistle blows. **Flagrant foul, type 2**—the referee makes the **ejection signal**. The coach **calls for a challenge**, the refs go to the **replay center**, review the **video evidence**... **call stands**. Player **ejected**, two **free throws** and **possession**."

NBA 比赛中裁判是隐形的主角。一个判罚可能改变比赛走向。这一篇把**犯规分类、违例规则、裁判手势、挑战系统**全部讲透。

## 规则体系 Rules & Officiating System

```
裁判 (Officials)
    ↓
犯规 (Fouls)
    ↓
违例 (Violations)
    ↓
判罚与挑战 (Calls & Challenges)
```

**犯规等级一览**：

> "Two **techs** in one game means **automatic ejection**."
> 一场比赛吃两次技犯自动驱逐。

`techs` 是 technical fouls 的口语简称。

### 教练挑战

| 英文                   | 音标                 | 中译     | 说明         |
| ---------------------- | -------------------- | -------- | ------------ |
| coach's challenge      | /ˈkoʊtʃɪz ˈtʃælɪndʒ/ | 教练挑战 | 教练申请回放 |
| call timeout           | /kɔːl/               | 叫暂停   | 挑战需要暂停 |
| signal for challenge   | /ˈsɪɡnl/             | 示意挑战 | 旋转手指手势 |
| challenge successful   | /səkˈsesfl/          | 挑战成功 | 改判         |
| challenge unsuccessful | /ʌnsəkˈsesfl/        | 挑战失败 | 维持原判     |
| call overturned        | /kɔːl ˌoʊvərˈtɜːrnd/ | 改判     | 推翻原判     |
| call stands            | /stændz/             | 维持原判 | 判罚不变     |
| call confirmed         | /kənˈfɜːrmd/         | 判罚确认 | 原判正确     |

> "Coach **calls timeout**, **signals for the challenge**."
> 教练叫暂停，示意挑战。

> "**Call confirmed**—the original ruling **stands**."
> 判罚确认——维持原判。

**Challenge 的三种结果**：
- **Overturned** = 改判（挑战成功）
- **Confirmed** = 确认（原判正确，回放清晰）
- **Stands** = 维持（回放不清晰，原判保留）

### 可挑战的判罚

| 类型                 | 是否可挑战 |
| -------------------- | ---------- |
| Personal foul        | ✅          |
| Out of bounds        | ✅          |
| Goaltending          | ✅          |
| Basket interference  | ✅          |
| Made vs. missed shot | ❌          |
| Traveling            | ❌          |
| Three seconds        | ❌          |

> "Coaches can challenge **fouls**, **out of bounds**, and **goaltending**, but not **traveling** or **three seconds**."
> 教练可以挑战犯规、出界和干扰球，但不能挑战走步或3秒。

> 他挥手取消进球——投篮前进攻犯规。

## Listen & Learn

一段完整裁判判罚解说：

> "Curry drives, contact... whistle. **Shooting foul on the three-point attempt**. But wait—Coach Kerr **calls timeout**, **signals for the challenge**. The refs go to the **monitor**, **review the play**... **call overturned!** It was a **clean play**, no foul. **Warriors retain timeout**, **Lakers' ball**. Big-time **challenge by Coach Kerr**."

拆解：
- **shooting foul** = 投篮犯规
- **calls timeout** = 叫暂停
- **signals for the challenge** = 示意挑战
- **call overturned** = 改判
- **clean play** = 干净的防守
- **retain timeout** = 保留暂停
- **big-time challenge** = 关键挑战

翻译：库里突破，有接触……哨响。三分投篮犯规。等等——科尔教练叫暂停，示意挑战。裁判走向显示器，回看判罚……改判！这是干净的防守，没有犯规。勇士保留暂停，湖人球。科尔教练的关键挑战。

## Practice 小练习

**Ⅰ. 犯规分类**

将犯规与类型配对：
1. 投篮时被推 → ____
2. 双方过度接触+不必要 → ____
3. 教练抗议被吹 → ____
4. 故意犯规罚球差的人 → ____

选项：technical foul / hack-a-Shaq / shooting foul / flagrant 2

**Ⅱ. 翻译**

1. He's in foul trouble with 5 fouls already.
2. Call confirmed—the original ruling stands.
3. He scores, gets fouled, converts the and-one—three-point play!

**Ⅲ. 填空**

1. Two techs in one game means automatic ____.
2. Coach's challenge can review fouls but not ____.
3. He's selling the ____—the refs aren't buying it.

---

**参考答案**

- Ⅰ. shooting foul / flagrant 2 / technical foul / hack-a-Shaq

- Ⅱ.
  1. 他陷入犯规麻烦，已经5次犯规。
  2. 判罚确认——维持原判。
  3. 他得分被犯规，加罚命中——3+1！

- Ⅲ. ejection / traveling / call
