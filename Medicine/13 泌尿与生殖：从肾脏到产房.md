# 13 泌尿与生殖：从肾脏到产房

## Opening Scene

凌晨三点的医院，两条走廊各有各的忙。一头是泌尿科病房，护士举着一袋淡黄色的引流液念叨："Output looks good, no blood this time."（尿量不错，这次没血了。）另一头是产房，一位准妈妈攥着丈夫的手，助产士在喊："One more push, the baby's crowning!"（再用一次力，宝宝的头快出来了！）

一边在排泄废物，一边在孕育新生——这两套系统在解剖学上离得很近，在医学里也常被合并成一个大科室来管。本篇我们就把这条"从肾脏到产房"的路走一遍。

泌尿系统是身体的污水处理厂，生殖系统是生命的孵化车间。两套系统的英文术语，一半来自希腊语，一半来自拉丁语，和我们前面几篇见过的规律一模一样。先认词，再拆词，最后你会发现验尿单和产检报告上那些吓人的长单词，其实都是积木拼出来的。

另外别被 uterus 和 hyster(o) 的关系绕晕：uterus 是器官的拉丁名（日常、解剖用），hyster(o) 是同义的希腊词根（造手术名用）。这种"一个器官两套词"的拉丁-希腊双轨制，下面的 Deep Dive 会专门拆。

## Deep Dive

### 1. nephr / ren：肾脏的希腊-拉丁双词根

肾脏在英文里有两副面孔。希腊词根是 **nephr(o)**，拉丁词根是 **ren(o)**。这又是我们在前几篇见过的老规律：**病名走希腊系，形容词走拉丁系。**

- 希腊系造病名：**nephritis** /nɪˈfraɪtɪs/ = nephr(o) 肾 + -itis 炎 = 肾炎；**nephrosis** = nephr(o) + -osis 异常状态 = 肾病；**nephrectomy** = nephr(o) + -ectomy 切除术 = 肾切除术。
- 拉丁系造形容词：**renal** /ˈriːnl/ = ren(o) + -al = "肾的"。所以"肾衰竭"标准说法是 **renal failure**，"肾功能"是 **renal function**。

记忆口诀：得病了喊希腊（nephritis），描述状态用拉丁（renal）。拆词公式见第 02 篇，词根总表见第 03 篇——这套"双词根"现象，在心脏（cardi-/cor-）、肺（pneumo-/pulmo-）上都出现过，肾脏只是又一个例子。

### 2. dialysis：透析就是"隔膜筛血"

**dialysis** /daɪˈæləsɪs/ 这个词看着唬人，拆开就明白了：

> **dia-**（穿过、贯穿）+ **-lysis**（分离、溶解）

合起来字面意思是"穿过去把东西分开"。临床上指的就是：让血液流经一层半透膜，膜的另一边是透析液，血里的废物（尿素、多余的水）"穿过"膜被分离出去，干净的血再流回身体。说白了，**透析就是隔着一层膜帮肾脏筛血**。

-lysis 这个后缀是个高产分子，凡是"溶解、分解"都用它：hemolysis（溶血，hemo 血）、electrolysis（电解）。记住 dia- + lysis 这个组合，dialysis 就再也不会拼错了。

### 3. OB/GYN：产科和妇科到底是什么科

医院里挂着 **OB/GYN** 牌子的科室，中文叫"妇产科"。这个缩写其实是两个词拼起来的：

- **OB = obstetrics** /ˌɒbˈstetrɪks/ 产科。这个词的拉丁词源 *obstetrix* 意思是"站在产妇面前的人"——也就是接生婆。产科管的就是怀孕、分娩、产后这一整段。专科医生叫 **obstetrician**。
- **GYN = gynecology** /ˌɡaɪnəˈkɒlədʒi/ 妇科 = **gynec(o)**（妇女）+ **-logy**（学）= "研究妇女（疾病）的学问"。专科医生叫 **gynecologist** = gynec(o) + -logist（专科医生）。妇科管的是女性生殖系统的非孕期问题。

把两者合起来，OB/GYN 就是"从怀孕到妇科病一手包办"的科室。顺带一提，gynec(o) 这个词根也藏在 misogyny（厌女，miso 恨 + gyny 女）里——医学词根经常和日常英语词撞脸。

### 4. -uria 后缀全家：验尿单上的词根游戏

体检验尿，化验单上一排以 **-uria**（…尿症）结尾的词，其实就是不同前缀拼上同一个后缀的填字游戏。-uria = ur(o)（尿）+ -ia（状态）：

| 英文          | 拆解                              | 中译       |
| ------------- | --------------------------------- | ---------- |
| hematuria     | hemat(o) 血 + -uria               | 血尿       |
| polyuria      | poly- 多 + -uria                  | 多尿       |
| oliguria      | oligo- 少 + -uria                 | 少尿       |
| dysuria       | dys- 困难/异常 + -uria            | 排尿困难   |
| proteinuria   | protein 蛋白 + -uria              | 蛋白尿     |
| anuria        | an- 无 + -uria                    | 无尿       |

看出门道了吗？只要认识前缀，-uria 家族就能一眼读懂：多了是 **polyuria**，少了是 **oliguria**，一滴没有是 **anuria**，撒尿费劲是 **dysuria**，尿里带血是 **hematuria**，漏了蛋白是 **proteinuria**。poly-（多）/ oligo-（少）/ an-（无）这组"数量前缀"和 dys-（异常）的释义，都对得上第 03 篇的词根总表。下次拿到验尿单，你就是自己的半个化验师。

## Read & Practice 实战

下面是一段虚构的产检对话，发生在产科诊室里。逐句拆给你看。

> **Doctor:** Your **due date** is March 10th, so you're about 32 weeks along now.
> **Patient:** Is everything okay with the baby?
> **Doctor:** The **ultrasound** looks great — the **fetus** is growing well. But your urine test shows some **protein in urine**, so we'll watch your blood pressure closely.
> **Patient:** Should I be worried about **delivery**?
> **Doctor:** Not yet. If labor doesn't start naturally by your due date, we'll discuss options, including a possible **C-section**.

逐句拆解：

- **"Your due date is March 10th, so you're about 32 weeks along now."** —— due date 预产期；32 weeks along 指"怀孕已经 32 周"，along 在这里是"进行到…程度"。医生在确认孕周。
- **"The ultrasound looks great."** —— ultrasound /ˈʌltrəsaʊnd/ = ultra-（超）+ sound（声）= 超声波检查，就是孕妇做的"B 超"。
- **"The fetus is growing well."** —— fetus 胎儿（拉丁词），比口语的 baby 更正式、更医学。
- **"Your urine test shows some protein in urine."** —— protein in urine 就是蛋白尿（proteinuria）的大白话版本。出现蛋白尿要警惕妊娠高血压，所以医生下一句提到 blood pressure（血压）。
- **"If labor doesn't start naturally... a possible C-section."** —— labor 这里指"产程、临产阵痛"；C-section 即剖宫产。医生在说：如果到预产期还没自然发动，就考虑剖。

这段对话把本篇的核心词几乎全用上了：due date、ultrasound、fetus、protein in urine、delivery、labor、C-section。读懂它，你就读懂了一次真实的产检。
