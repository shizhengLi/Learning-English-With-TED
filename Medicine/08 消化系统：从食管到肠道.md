# 08 消化系统：从食管到肠道

## Opening Scene

体检中心，你刚做完无痛胃镜，护士递来一张报告单。你还没从麻醉的迷糊里完全醒过来，眼睛先撞上了最上面那行加粗的英文：

> "Endoscopy findings: mild gastritis, no ulcer detected."

你认得 no，认得 detected，可中间那两个长词像两块拦路石。gastritis？ulcer？是好是坏？要不要紧？医生在旁边淡淡补一句"轻度胃炎，没事"，你这才松口气——但心里那点不踏实还在：明明每个字母都认识，连起来怎么就成了天书。

其实那位医生没背过这个词，他是**拆**着读的：gastr(胃) + -itis(炎)，胃 + 发炎 = 胃炎。拆词公式见第 02 篇，一旦你把消化系统这套"乐高积木"摸熟，下次再拿到这种报告单，你扫一眼就知道哪行是器官、哪行是症状、哪行是医生最关心的结论。

这一篇，我们就从口腔一路走到肛门，把这条人体里最长的"管道"沿途的英文标牌一块块认下来。

## 核心词汇 Core Vocabulary

消化系统的英文有个特别友好的地方：它走"双轨制"。日常器官名（liver、stomach）是好懂的盎格鲁-撒克逊小词，而病名和手术名则换上希腊词根（hepat、gastr）。认全这两套，报告单就拦不住你了。

## Deep Dive

### 一、GI tract：从口到肛，其实是一条管

医院里医生张口闭口的 **GI**，全称是 **gastrointestinal**，读作 /ˌɡæstroʊɪnˈtestɪnl/。拆开看一目了然：

> **gastro(胃) + intestinal(肠的) = 胃肠的**

加上 **tract**（/trækt/，意为"道、束"），**GI tract** 就是"胃肠道"。但这个名字其实有点谦虚——它远不止胃和肠。完整的这条管子，从上到下依次是：

口腔(mouth) → 食管(esophagus) → 胃(stomach) → 小肠(small intestine) → 大肠(large intestine) → 直肠(rectum) → 肛门(anus)。

把它想象成一根贯穿身体的水管，食物从一头进、残渣从另一头出，全程被这根管子加工。而 liver(肝)、gallbladder(胆囊)、pancreas(胰腺) 是挂在管子旁边的"附属工厂"，负责往管腔里分泌胆汁和消化酶，它们合称 **accessory organs**（附属器官）。所以当你看到 **upper GI**（上消化道）和 **lower GI**（下消化道）这种说法，分界点大致就在小肠中段——上消化道出血和下消化道出血，处理思路完全不同。

记住 GI 这两个字母，你就握住了整个消化科的"门牌总号"。

### 二、heartburn：烧的根本不是心

新人最容易被坑的词，非 **heartburn** 莫属。字面 heart(心) + burn(烧)，听着像心脏出了急事，吓得人想挂心内科。但真相是——**它跟心脏一点关系都没有**。

烧心的本质是胃酸反流到食管，灼烧食管下段的黏膜，只因为这片区域恰好在心脏后方，古人就误以为是"心在烧"。这个俗名于是讲三层进阶，越往下越"医学"：

1. **heartburn**（烧心）——病人的主诉，厨房水平的大白话。
2. **acid reflux**（/ˈæsɪd ˈriːflʌks/，胃酸反流）——描述机制：acid(酸) + reflux(re-倒 + flux 流 = 倒流)，胃酸往上倒流。
3. **GERD**（胃食管反流病）——正式诊断，全称 **gastroesophageal reflux disease** /ˌɡæstroʊɪˌsɑːfəˈdʒiːəl/。再拆一次：gastro(胃) + esophageal(食管的) + reflux(反流) + disease(病)。

同一件事，病人说 heartburn，医生写 GERD。你能在三层之间自由翻译，就算真正听懂了门诊对话。顺带一提，GERD 里那个吓人的长词 gastroesophageal，正是"胃 + 食管"两个词根拼起来的，拆词公式见第 02 篇，再长也不怕。

### 三、-scopy 镜检家族：在消化科大放异彩

后缀 **-scopy**（镜检）在消化科简直是顶流。原理都一样——把一根带摄像头的软管伸进体内直接看，区别只在"看哪儿、从哪进"。三个高频词构成一组漂亮的包含关系：

- **endoscopy**（内镜检查）= endo-(内) + -scopy。这是**总称**，泛指一切"伸镜子进体内看"的操作。
- **gastroscopy**（胃镜）= gastr(胃) + -scopy。从口腔进，看食管、胃、十二指肠，属于上消化道内镜，所以也叫 **upper endoscopy**。
- **colonoscopy**（结肠镜）= colon(结肠) + -scopy。从肛门进，看大肠全程，属于下消化道内镜。

换句话说，gastroscopy 和 colonoscopy 都是 endoscopy 这个大家族里的具体成员——一个管上头，一个管下头。把这条 -scopy 流水线记牢，你会发现它在别的系统里照样高产：bronchoscopy(支气管镜)、arthroscopy(关节镜)……认准后缀，批发整排单词。

### 四、liver 与 hepat 的双轨，以及 hepatitis A/B/C

最后回到那个"双轨制"。"肝"这个器官，日常叫 **liver**：肝功能检查写作 **liver function test**（缩写 **LFT**），抽血查的就是它。可一旦肝**发炎**，名字立刻切换成希腊词根 **hepat**：hepat(肝) + -itis(炎) = **hepatitis**(肝炎)。

为什么不直接说 liver-itis？因为医学病名几乎全部沿用希腊/拉丁词根（这段历史第 01 篇讲过）。所以你要同时记住这对"双轨"：**器官用 liver，疾病用 hepat**。同理，肝脏专科医生叫 **hepatologist**（hepat + -logist 专科医生），研究肝病的学科叫 **hepatology**。

而肝炎后面那串 A/B/C 字母，命名逻辑朴素得可爱——就是**按病毒被发现的先后顺序编号**：

- **Hepatitis A**（甲肝）：粪口传播，吃坏东西可能中招，多半能自愈。
- **Hepatitis B**（乙肝）：血液和体液传播，可转慢性，有疫苗可防。
- **Hepatitis C**（丙肝）：主要经血液传播，过去难治，如今已有特效药。

A、B、C 不代表轻重，只代表"第几个被人类认识"。记住 hepat 这个词根，再配上字母编号，一整组肝炎术语就被你打包带走了。

## Read & Practice 实战

下面是一份虚构的胃镜检查报告，我们逐行拆给你看。真实报告的措辞和这个八九不离十：

> **Endoscopy Report**
> 1. Esophagus: normal, no reflux esophagitis observed.
> 2. Stomach: mild gastritis, mucosa slightly red.
> 3. No ulcer or mass detected.
> 4. Duodenum: unremarkable.
> 5. Impression: mild gastritis. Recommend follow-up.

逐句拆解：

- **第 1 行**：Esophagus(食管) normal(正常)，no reflux esophagitis observed——没观察到反流性食管炎。reflux(反流) + esophagitis(食管炎，esophag 食管 + -itis 炎)，连起来就是上面 Deep Dive 讲的 GERD 在食管留下的炎症，这里"没有"，好消息。
- **第 2 行**：Stomach(胃) mild gastritis——轻度胃炎。mucosa(/mjuːˈkoʊsə/，黏膜) slightly red——黏膜略红，这是胃炎的典型表现。
- **第 3 行**：No ulcer or mass detected——未发现溃疡或肿块。ulcer(溃疡) 和 mass(肿块) 是医生最警惕的两样，这行说"都没有"，是整张报告最让人安心的一句。
- **第 4 行**：Duodenum(/ˌduːəˈdiːnəm/，十二指肠，词根 duoden-) unremarkable——这个词字面"无可记述"，在报告里是行话，意思就是"没毛病、一切正常"。
- **第 5 行**：Impression(印象/结论) 是报告的"判决书"，浓缩成一句 mild gastritis(轻度胃炎)；Recommend follow-up——建议复查随访。

读完五行你会发现：报告的信息密度极高，但句式高度套路化。认得器官词根 + 认得 -itis/ulcer/mass + 认得 normal/unremarkable 这几个"正常黑话"，一份胃镜报告就被你读穿了。

## Practice 练习

**Ⅰ. 拆词（拆成词根/词缀并写出中文）**

1. hepatomegaly /ˌhepətoʊˈmeɡəli/
2. cholecystectomy /ˌkoʊləsɪsˈtektəmi/
3. dysphagia /dɪsˈfeɪdʒə/
4. gastroenteritis /ˌɡæstroʊˌentəˈraɪtɪs/

---

**Ⅱ. 症状中译英（用本篇核心词）**

1. 我这两天恶心、没胃口。
2. 他饭后总是烧心、腹胀。
3. 她拉肚子拉了三天了。

---

**Ⅲ. 把下列消化道器官按"食物经过的先后顺序"排列**

> rectum, stomach, esophagus, large intestine, small intestine

---

**参考答案**

- **Ⅰ.**
  1. hepat(肝) + o + -megaly(肿大) → **肝肿大**
  2. cholecyst(胆囊，chole 胆 + cyst 囊) + -ectomy(切除术) → **胆囊切除术**
  3. dys-(困难) + -phagia(吞咽) → **吞咽困难**
  4. gastr(胃) + o + enter(肠) + -itis(炎) → **胃肠炎**（一个词里塞了两个器官词根，正是拆词法的威力）

- **Ⅱ.**
  1. I've felt nauseous and have no appetite these two days.（恶心的形容词形式是 nauseous /ˈnɔːʃəs/）
  2. He always gets heartburn and bloating after meals.
  3. She has had diarrhea for three days.

- **Ⅲ.** esophagus(食管) → stomach(胃) → small intestine(小肠) → large intestine(大肠) → rectum(直肠)。这正是 GI tract 从上到下的顺序。
