# 04 必记前缀后缀：从hyper-到-ectomy

## Opening Scene

> "Patient's tachycardic, heart rate 145. BP dropping. We need to move."

凌晨三点的急诊室，监护仪的报警声尖锐刺耳。住院医盯着屏幕念出一个词：**tachycardia**（心动过速，/ˌtækiˈkɑːrdiə/）。心率 145，太快了。

如果监护仪报的是另一个词——**bradycardia**（心动过缓，/ˌbreɪdiˈkɑːrdiə/），心率降到 38——那是完全相反的危机：心脏跳得太慢，泵不出足够的血。

`tachy-`（快）和 `brady-`（慢），只差几个字母，方向却完全相反，处理方案天差地别。在医学英语里，**前缀决定方向，后缀决定动作，词根决定部位**。词根你在第 03 篇已经见过一张总表了；这一篇，我们把"决定方向和动作"的前缀、后缀一次性铺开。读完这篇，你看到一个十几个字母的长词，也能像拆乐高一样把它拆开、读懂。

## Deep Dive

### 1. hyper / hypo 全家福：前缀一换，方向反转

`hyper-`（过高）和 `hypo-`（过低）是医学里最爱成对出现的一对前缀。它们后面接同一个词根，意思立刻南辕北辙。看这三组对照：

| hyper-（高） | hypo-（低） |
| --- | --- |
| hypertension 高血压 | hypotension 低血压 |
| hyperglycemia 高血糖（glyc 糖 + emia 血症） | hypoglycemia 低血糖 |
| hyperthyroidism 甲状腺功能亢进 | hypothyroidism 甲状腺功能减退 |

注意读音陷阱：`hyper-` 读 /ˈhaɪpər/，`hypo-` 读 /ˈhaɪpoʊ/，区别全在词尾那个元音——一个是模糊的 /ər/，一个是饱满的 /oʊ/。临床上听错一个音，"亢进"就变成"减退"，意思完全反了。读这两个前缀时务必把词尾咬清楚。

### 2. 手术四件套：-ectomy / -otomy / -ostomy / -plasty

这四个后缀长得像，但代表四种完全不同的操作。记住一句口诀：**切掉、切开、造口、整形**。

- **-ectomy 切除术** = 把器官整个或部分拿走。`appendectomy`（阑尾切除术）就是把发炎的阑尾切掉。
- **-otomy 切开术** = 只切开、不取走。`tracheotomy`（气管切开术）是在气管上开个口让病人呼吸，气管还在。
- **-ostomy 造口术** = 造一个永久或长期的开口通到体外。`colostomy`（结肠造口术）把结肠引到腹壁外，接一个袋子收集排泄物。
- **-plasty 成形术** = 修复或重塑外形。`rhinoplasty`（鼻成形术，rhin 鼻）就是大家常说的"隆鼻 / 鼻整形"。

辨别窍门看中间那几个字母：`-ec-`（切除）、`-ot-`（切开）、`-ost-`（造口，比 -ot- 多个 s）、`-plast-`（塑形，和 plastic 同源）。

### 3. -itis vs -osis vs -pathy：炎症、状态、泛指病变

这三个后缀都能跟在器官词根后面表示"那里出问题了"，但精确程度不同：

- **-itis 炎症**：明确是炎症反应，通常伴红肿热痛。`gastritis` 胃炎、`arthritis` 关节炎、`hepatitis` 肝炎。看到 -itis 就想到"发炎了"。
- **-osis 异常状态**：泛指一种异常的状态或退行性改变，不一定是炎症。`nephrosis` 肾病、`fibrosis` 纤维化。它更"中性"，只是说"处于某种异常状态"。
- **-pathy 病变**：最笼统，"那个部位有病变，但没说清是哪种"。`neuropathy` 神经病变、`cardiomyopathy` 心肌病。医生还没定性时常用它。

一句话：**-itis 最具体（炎症），-osis 居中（异常状态），-pathy 最模糊（泛指病变）**。

### 4. a- / an- 的隐身术：否定前缀藏在词里

否定前缀 `a-`（辅音前）和 `an-`（元音前）非常会"伪装"，它们和词根黏在一起后，初学者经常看不出来这是个否定词。拆给你看：

- **anemia** = an（无）+ emia（血症）→ 字面"无血"，实际指血红蛋白不足，即**贫血**。词根 emia 以元音开头，所以用 an-。
- **apnea** = a（无）+ pnea（呼吸）→ "无呼吸"，即**呼吸暂停**。pnea 以辅音音开头，所以用 a-。
- **anuria** = an（无）+ uria（尿）→ "无尿"，**无尿症**（注意和上面 oligo- 的"少尿" oliguria 区分：少 ≠ 无）。

记住：碰到以 a- / an- 开头、又拆不出意思的医学词，先试着把它当"否定"理解，往往豁然开朗。

## Read & Practice 实战

下面是一段虚构的急诊交接班口头报告（handoff）。住院医语速飞快，词缀词一个接一个——我们逐句拆开看。

> "Bed 4 is a 22-year-old male, **post**-**appendectomy** day one. Overnight he became **tachycardic** to 130 and **hypotensive**, BP 85 over 50. We were worried about **hemorrhage**, so we ran an **angiography** — no active bleeding. **Gastroenterology** is consulting. He's also **dyspneic**, but no signs of **pneumonia** on the chest film."

逐句拆解：

- **post-appendectomy day one** = post（后）+ appendectomy（阑尾切除术）→ "阑尾切除术后第一天"。一个词同时用上了时间前缀和手术后缀。
- **tachycardic to 130** = tachy（快）+ cardi（心）+ ac（形容词后缀）→ "心动过速到 130"。tachycardia 的形容词形式。
- **hypotensive**, BP 85 over 50 = hypo（过低）+ tens（压）→ "低血压"，血压 85/50，确实偏低。
- worried about **hemorrhage** = hem（血）+ rrhage（出血）→ 担心"大出血"。
- ran an **angiography** = angi（血管）+ graphy（造影）→ 做了"血管造影"看有没有出血点。
- **Gastroenterology** is consulting = gastr（胃）+ enter（肠）+ logy（学）→ "消化内科"来会诊。
- He's **dyspneic** = dys（困难）+ pne（呼吸）→ "呼吸困难"。
- no signs of **pneumonia** = pneumon（肺）+ ia（状态）→ 胸片上没有"肺炎"迹象。

看懂了吗？整段话里至少八个词是用前缀 + 词根 + 后缀拼出来的。掌握了这套规则，再快的交接班你也能跟上。

## Practice 练习

**Ⅰ. 前缀反义配对**：把下列前缀和它的反义前缀连起来。

1. hyper-
2. tachy-
3. pre-
4. poly-
5. endo-（提示：和"周围/外侧"相对）

候选：post- / oligo- / hypo- / peri- / brady-

**Ⅱ. 词缀组合：给词缀造词或解词**

1. "肝肿大"怎么拼？（hepat 肝 + 肿大后缀）
2. gastroscopy 是什么意思？逐部分拆解。
3. "白细胞减少"用哪个后缀？写出完整词。

**Ⅲ. 中译英**

1. 病人术后出现低血糖。
2. 我们安排了一次结肠镜检查（colon 结肠 + 镜检后缀）。
3. 心电图显示心动过缓。

---

**参考答案**

- Ⅰ.
  1. hyper- ↔ hypo-（过高 ↔ 过低）
  2. tachy- ↔ brady-（快 ↔ 慢）
  3. pre- ↔ post-（前 ↔ 后）
  4. poly- ↔ oligo-（多 ↔ 少）
  5. endo- ↔ peri-（内 ↔ 周围）

- Ⅱ.
  1. **hepatomegaly**（hepat + o + megaly）肝肿大。
  2. gastro（胃）+ scopy（镜检）= **胃镜检查**。
  3. 用 **-penia**（减少）→ **leukopenia**（leuk 白 + o + penia）白细胞减少。

- Ⅲ.
  1. The patient developed **hypoglycemia** postoperatively.（或 ... after the operation）
  2. We scheduled a **colonoscopy**.
  3. The **electrocardiogram** (ECG/EKG) showed **bradycardia**.
